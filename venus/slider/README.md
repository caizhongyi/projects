alienjs-slider
===============

A alienjs-slider.js plugin


Install
-------

Install with npm:

    $ npm install alienjs-slider

Usage
-----

```js
  $.fn.slider({
        interval : 5000 ,
        auto     : true ,
        easing   : 'linear',
        duration : 'normal',
        page    : 0,
        index    : 0
  });
```

For more details please visit [中文文档](https://www.alienjs.net)
