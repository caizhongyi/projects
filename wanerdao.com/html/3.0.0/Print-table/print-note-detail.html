中新社华盛顿12月5日电 (记者 德永健) 在对往事的回忆中，美国总统奥巴马5日缅怀过世的南非国父曼德拉。他说，曼德拉激励自己开启了政治生涯。 　　华盛顿时间5日下午5时20分，曼德拉过世的消息传出后不久，奥巴马在白宫发表讲话，第一时间向享年95岁的曼德拉致敬。 　　奥巴马说，与很多人一样，如果没有曼德拉树立的榜样，“我完全无法想象自己的生活”，只要还活着，他会一直向曼德拉学习。 　　奥巴马回忆，在曼德拉的激励下，他开启了政治生涯，“我第一次参与和政治有关的事情，就是抗议南非种族隔离”。 　　“我研读过他的讲话和著作。他出狱的那一天，让我明白人类用希望而不是用恐惧指引自己时可以做到的事情。”奥巴马说。 　　在白宫讲话中，奥巴马颂扬曼德拉是世上最有影响力、最有勇气的人，并且非常友善，风趣幽默，可以与任何人相处。 　　奥巴马特别指出，身为南非国父，曼德拉却不自满，勇于承认自己的不足。他援引曼德拉的话说，“我不是圣人，除非圣人是位不断尝试的罪人”。 　　此外，奥巴马说他和夫人米歇尔向曼德拉的家人致以最深的敬意，希望曼德拉最后与家人共度的时刻，为他们带来安宁和慰藉。 　　对于南非人民，奥巴马称曼德拉一直梦想南非实现民主和自由，如今南非复兴，种族和解，为世界树立榜样，曼德拉可谓给南非留下丰功伟绩。 　　“现在，让我们停下工作感谢曼德拉在世，他亲手缔造历史，让世间的道德转向正义，愿上帝铭记他，保佑他安息。”奥巴马祈祷道。 　　作为同样创造历史的黑人总统，奥巴马一直将曼德拉尊为前辈，视作偶像。2005年，刚刚就任参议员的奥巴马第一次与曼德拉见面，2009年，曼德拉给入主白宫的奥巴马写过信，后来两人偶尔会通过电话交谈。 　　2011年米歇尔访问南非，曾带女儿探望曼德拉，不过今年6月奥巴马亲自访问南非，因曼德拉病重入院，两人未能见面，奥巴马只是走访了当年曼德拉被囚禁的监狱；11月初，白宫曾放映讲述曼德拉生平的传记片，当时曼德拉的两位女儿受邀与奥巴马一家观看了影片。(完)
