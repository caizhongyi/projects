    周末组团唱歌(2011/11/11)

    我们在极端情境下，第一反应脱出而出的应该是母语，香港人急了怒了都会说丢，北方从说操或靠，上海人说册那。我有一阵看多了美国片，极端情绪下会说shit和fuck,这反应装逼到自己很不舒服，强行改掉，恢复成字正腔圆的册那。今晚被猫叫吓到时，脱口而出的居然是陕西话：饿贼尼玛！看你还是不够怕。

    不在极端情况也一样，我有两个同事一个广东人在武汉读四年书毕业后回广东，无论是聊天还是开会每句话必带个尾音：个板马的；另一湖南人在北京读四年书，无论是聊天还是开会，每句话后面捎上那句举世闻名的“我操”。
