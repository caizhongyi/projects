太原

九种食物改善失眠

分类名

从网页游戏的角度看《龙将》，其可玩性还是相对比较高的！游戏中声望、将魂和银币是玩家增强实力的保障，新手平民（包括不充值和VIP1玩家）如何从新手迅速壮大呢？在这里本人浅谈下个人经验！

进入新区，冲级是最重要的，接任务做任务快速升到30级，因为30级的装备属性决定了太多东西！而进入新区前，准备工作一定要做足！几乎每开新区会发很多的新手卡，而不同的新手卡可以同时使用的，在这里推荐下官网新手卡+256G+07073，一般情况下这三个网站发的新手卡都是可以同时使用的，而且有的网站会发几个不同的礼包，如白金新手包，钻石新手包等，先领了进游戏后激活，会得到不少东西，这些白得的东西对任何玩家都是有点小作用的！

武将招募：50级之前是攒钱的阶段，注意哦，一定要注意，招募武将是很费钱的，VIP1玩家以下建议在30的时候只招募黄盖和徐庶即可，这俩武将虽然是蓝色的，如果装备和强化给力的话，50级以后还是能用很长时间的。50级的时候用剩下的钱开始打造属于自己的军团，VIP1玩家用积累的银币开始高级对酒。

注意：猜拳是很有讲究的，注意武将说的话，第一轮武将说的话出的招数和以后武将说的话相同，则出的招数是一样的。
