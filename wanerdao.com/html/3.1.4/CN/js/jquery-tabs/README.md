alienjs-tabs
===============

A alienjs-tabs.js plugin


Install
-------

Install with npm:

    $ npm install alienjs-tabs

Usage
-----

```js
  $.fn.tabs({
        list : '.tabs-list',
        target  : '[data-target]',
        event : 'click'
  });
```

For more details please visit [中文文档](https://www.alienjs.net)
