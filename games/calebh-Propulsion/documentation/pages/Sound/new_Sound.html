new Sound

new Sound(url);
url: [string] The url the sound should be loaded from.

[object] A new SOUNDOBJECT.

aud.bgmusic = new Sound('sound/bgmusic.ogg');

Note: The sound doesn't actually begin loading until the load function is called (see Loading & Initialization)

Make sure you use an sound format supported across browsers.

A SOUNDOBJECT has all of the features that an object created with new Audio() has.
