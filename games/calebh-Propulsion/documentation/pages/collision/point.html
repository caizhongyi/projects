collision.point

collision.point(obj,px,py,enableMTV);

o1: [object] The object to check for a collision.

px: [number] Horizontal position of the point to check.

py: [number] Vertical position of the point to check.

enableMTV: (Optional) [boolean] If true, the objects will be moved along the minimum distance possible for collision to no longer occur.

[boolean] True will be returned if the point is within the object, and false will be returned if it is outside the object.

See also: the mask and angle object properties.
