collision.rectangle

collision.rectangle(obj,rx,ry,width,height,enableMTV,enableCPoint);

obj: [object] The object to check for a collision.

rx: [number] The horizontal position of the upper left point of the rectangle.

ry: [number] The vertical position of the upper left point of the rectangle.

width: [number] The width of the rectangle to check. It can be a negative number.

height: [number] The height of the rectangle to check. It can be a negative number.

enableMTV: (Optional) [boolean] If true, the object will be moved the minimum distance possible for collision to no longer occur.

enableCPoint: (Optional) [boolean] If this parameter is true, then the point at which the rectangle and the object touch after the MTV has been applied will be returned in an object. The returned object contains x and y properties that store the collision point. Note that the enableMTV parameter must be true for the contact point to be returned.

[boolean] or [object]. If enableMTV and enableCPoint are true, and the rectangle and object are colliding, then an object containing the contact point of the object and the rectangle after the MTV has been applied will be returned.

Otherwise, a value of true will be returned if the objects are colliding.

A value of false will be returned if the objects are not colliding.

See also: the mask and angle object properties.
