Detecting Key Presses

The key object contains many objects that store the state of each key. Each of these objects contains a down, pressed, and up variable with a boolean value. 
// The layout of all the key objects
var key = {
	KEYNAME = {
		down: [boolean], // Key was pressed down since the end of the last tick
		pressed: [boolean], // Key is currently down
		up: [boolean] // Key was released since the end of the last tick
	}
};
The available keys are:
  • Letters (a-z)
  • 0-9
  • space
  • enter
  • tab
  • escape
  • backspace
  • shift
  • control
  • alt
  • capsLock
  • numLock
  • left
  • right
  • up
  • down
  • insert
  • del
  • home
  • end
  • pageUp
  • pageDown
  • f1-f12
if (key.a.down) {
	alert('a pressed');
}

if (key.space.pressed) {
	alert('space bar is currently pressed');
}

if (key.f3.up) {
	alert('f3 key released');
}

if (key[3].down) {
	alert('3 key pressed');
}
