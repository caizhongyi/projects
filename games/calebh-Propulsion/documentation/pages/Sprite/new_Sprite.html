new Sprite

new Sprite(url,subimg,xorig,yorig);

url: [string] The url to load the sprite from

subimg: [number] The number of subimages in the image (So that sprite strips can be seperated)

xorig: [number] The location to set the sprite's origin; Also the center of rotation

yorig: [number] The location to set the sprite's origin; Also the center of rotation

[object] The newly created SPRITEOBJECT. 
spr.ball = new Sprite('sprites/ball.png',2,15,15);

Note: The sprite doesn't actually begin loading until the load function is called.
