SPRITEOBJECT.draw

SPRITEOBJECT.draw(x,y,subimg,angle,hScale,vScale);

x: [number] x coordinate to draw the sprite at

y: [number] y coordinate to draw the sprite at

subimg: (Optional) [number] the subimage you wish to draw (for sprite strips). 0 is the first subimage. If a decimal is passed, the floor of the passed value is used.

angle: (Optional) [number] angle in radians to rotate the sprite

hScale: (Optional) [number] The horizontal scale factor to apply. If this number is negative, the sprite will be drawn flipped horizontally.

vScale: (Optional) [number] The vertical scale factor to apply. If the number is negative, the sprite will be flipped vertically.

[object] The SPRITEOBJECT 
spr.ball = new Sprite('sprites/ball.png',2,15,15);
spr.ball.draw(122,253,1,0);
