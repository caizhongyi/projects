draw.text

draw.text(x,y,text,stroke,maxWidth);

x: [number] The x coordinate to draw the text at

y: [number] The y coordinate to draw the text at

text: [string] The text to draw to the screen

stroke: [boolean] (Optional) If true, the font should be stroked instead of filled. (Default false)

maxWidth: [number] (Optional) If a maximum width is provided, the text will be scaled to fit that width if necessary.

draw.text(167,184,'foobar');
