draw.rectangle

draw.rectangle(x1,y1,width,height,stroke,color);

x1: [number] The x coordinate of the first point

y1: [number] The y coordinate of the first point

width: [number] The width of the drawn rectangle in pixels

height: [number] The height of the drawn rectangle in pixels

stroke: [boolean] (Optional) If true, a stroked rectangle should be drawn (default is false)

color: [string] (Optional) A string with a css color. Sets the drawing color.

draw.rectangle(12,15,80,67,true);
