draw.triangle

draw.triangle(x1,y1,x2,y2,x3,y3,stroke,color);

x1: [number] The x coordinate of the first point

y1: [number] The y coordinate of the first point

x2: [number] The x coordinate of the second point

y2: [number] The y coordinate of the second point

x3: [number] The x coordinate of the third point

y3: [number] The y coordinate of the third point

stroke: [boolean] (Optional) If true, a stroked rectangle should be drawn (default is false)

color: [string] (Optional) A string with a css color. Sets the drawing color.

draw.triangle(19,14,31,52,64,29,false,'pink');
