draw.circle

draw.circle(x,y,radius,color);

x: [number] The x coordinate to draw the circle

y: [number] The y coordinate to draw the circle

radius: [number] The radius in pixels to draw the circle

stroke: [boolean] (Optional) If true, the circle should be stroked instead of filled. Default is false (filled circle)

color: [number] (Optional) Sets the drawing color

draw.circle(784,430,50,false,'blue');
