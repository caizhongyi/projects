draw.depth

draw.depth(obj,depth);

obj: [object] The object to set the depth of.

depth: [number] The depth to set the object to.

Registered objects with a higher depth property have their methods invoked last. Objects with a higher depth property will thus most likely have their draw functions invoked last, and will appear "above" other drawings.

Note: The new depths will only take effect after the currently running set of methods have finished running.
