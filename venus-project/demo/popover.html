参数列表
参数名 参数类型 默认值 说明
target string .popover-target 触发事件的标签
position string l（l | b | r | t） 默认显示在目标标签的位置
offset object { left : 0, top : 0} 偏移的位置
duration string|number normal 切换间隔
easing string easeInOutExpo 赛尔曲线
event string click 响应事件
init function window['popoverInit'] 初始化后的回调
事件列表
事件名 说明
shown popover.on('shown' ,function( e, args){})
hidden popover.on('hidden' ,function( e, args){})
