aliencss-common
===============

A aliencss-common.css plugin


Install
-------

Install with npm:

    $ npm install aliencss-common

Usage
-----

```css
     //文本
      .text-bold
      .text-danger,.text-important
      //淡红色
      .text-success
      //淡绿色
      .text-warning
      //淡黄色
      .text-muted
      //淡灰色
      .text-info
      //淡蓝色
      .text-default
    
      .text-xxxlg
      .text-xxlg
      .text-xlg
      .text-lg
      .text-md
      .text-sm
      .text-xs
    
      .text-ellipsis
    
      .text-hide
    
      .text-right
      .text-left
      .text-center
      .text-justify
      .text-indent 
```

For more details please visit [中文文档](https://www.alienjs.net)
